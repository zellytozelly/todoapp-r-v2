# TodoApp - react
202112714 김수진

### React.js를 이용한 할 일 관리 앱
---
## 시작하기

### `yarn start`

브라우저 확인 : [http://localhost:3000](http://localhost:3000)

---
## 컴포넌트 구성

### `Add` : 할 일 추가
### `Front` : 할 일 통계 (남은 할일/전체 할일)
### `Layout` : 전체적인 레이아웃
### `List` : 할 일 리스트
### `Todo` : 각 할 일. 완료/미완료 상태 설정
